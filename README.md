# Job Tracker

A simple and clean job application tracker built with React, Vite, and Tailwind CSS.  
This project focuses on solid React fundamentals such as component design, custom hooks, and state management.

## Features
- Add and delete job applications
- Track application status
- Persistent data using localStorage
- Clean and scalable component structure

## Tech Stack
- React
- Vite
- Tailwind CSS

## What I Learned
- Thinking in components
- Lifting state and data flow
- Controlled forms
- Custom hooks (`useJobs`)
- Side effects with `useEffect`
- LocalStorage persistence
